# Python Banking System

A structured command-line banking system built using Python, designed to simulate core banking operations while focusing on clean code practices, system design, and scalability.

---

## 🚀 Features

- Account creation with validation
- Secure login system (Password + MPIN)
- Deposit and withdrawal functionality
- Account management:
  - Change password
  - Change MPIN
  - Delete account
- Money transfer between accounts
- Transaction history tracking
- Input validation and error handling
- Account lock system for failed attempts
- Secure password storage (hashing)
- Database integration for persistent storage

---

## 🧠 Purpose of the Project

This project is built to:

- Strengthen Python fundamentals
- Practice writing clean and maintainable code
- Understand real-world system design concepts
- Learn authentication, security, and data handling
- Transition from basic scripting to structured development

---

## 🛠️ Tech Stack

- Python
- File Handling → transitioning to SQLite database
- Hashing (for security)
- CLI-based interface (planned upgrade to GUI)
